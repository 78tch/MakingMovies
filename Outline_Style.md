# Wordを使いこなすための３つの基本を押さえよう
1. 「アウトライン」  
Wordは、本来、目次や索引を付けるような長文、本や論文のような長文を書くためのソフト（アウトラインプロセッサ）です。  
長文を書くとき、いきなり本文を書き始めてしまうと、うまく考えをまとめられないために、話題が重複したり、書き洩らしたりしてしまって、読みづらい文章になってしまいがちです。  
これを避けるためには、まず文書の全体構成を構想し、「章・節・項」のように章立てを書き出してから、そのあとに各章の本文を書く、という手順をとります。  
このとき、推敲の手助けをしてくれるWordの便利な機能が、「見出し行」に「最大で9段階のレベル付け」をすることができる、「アウトライン」機能です。  
ただし一番下の「レベル９」まで使うことはめったになくて、通常は「章・節・項」として「レベル１」から「レベル３」までを使えば充分であることが多いでしょう。  
アウトライン機能の使い方は、Wordを「印刷レイアウト」表示から「アウトライン」表示に切り替えて、文書中の「見出し行」に「レベル付け」をすることで、文書を「見出し行」の「レベル」に応じた、入れ子状のツリー表示にすることができます。  
これにより、文書の全体構成や現在の編集位置を把握しながら文章を記述することができますし、さらには、見出し行のレベルを「章・節・項」の間で上下させたり、前後を入れ替えたりといった移動も簡単にできます。  
このように、Wordの「アウトライン」機能を使いこなせば、章・節・項ごとの話題の「粒度」や「階層の深さ」を揃えて、「読みやすい順序」に並べることができます。  
「アウトラインプロセッサ」は、効率のよい推敲を助けてくれるツールなのです。  
ちなみに、こうして適切にアウトラインが設定された文書のことを「構造化文書」と呼びます。  
2. 「スタイル」  
文書の見栄えをよくするために、フォントや文字サイズなどの書式設定をすると思います。  
このうち、特に見出し行については、文字サイズを本文より大きくしたり、太字にしたり、本文とは別のフォントにしたり、水平線（罫線）を引いたりといった、修飾的な書式設定をすると思います。  
その際、デザインに統一感を持たせるために、同一レベル（階層）の「見出し」には、同じ書式設定を適用するべきでしょう。  
しかしそのすべてを手作業でやっていたのでは、面倒ですし、作業漏れが発生しかねません。  
そこで、こういった「複数項目にわたる書式設定のワンセット」に、名前を付けて保存して、あとで呼び出せるようにできます。  
これが「スタイル」という機能で、指定した箇所に、ワンセットの書式設定をまとめて適用することができます。  
さらに、前もってアウトラインの特定の「レベル」に特定の「スタイル」を紐づけしておくことによって、文書で「レベル付け」をすれば、各見出しに自動的に「スタイル」が適用されるようにできます。  
例えば、「章」である「レベル１」に「見出し１」スタイルを、「節」である「レベル２」に「見出し２」を、「項」である「レベル３」に「見出し３」スタイルを紐づける、ということをします。  
その方法は、例えば「見出し１」スタイルにて->「変更」->「書式」->「段落」->「インデントと行間隔」->「全般：アウトラインレベル」で「レベル１」などと指定します。  
これによって、文書にレベル付けをすれば、行見出しに「見出し１」などのスタイルが自動的に適用されます。  
「スタイル」において、例えば「見出し１」「見出し２」「見出し３」の書式設定を詰めておくだけで、デザインに統一感のある見出しが簡単に作れるわけです。
なお、スタイルには、適用する範囲に応じて「文字（フォント）」、「段落」、「リンク（段落と文字）」の3種類があります。  
このうち、「文字スタイル」の「フォントサイズ」、「段落スタイル」の「間隔」（段落前、段落後、行間）が、１ページに収まる行数に関わってくる部分です。  
新規文書に特に指定をしないと、「アウトライン」のレベルは「本文」が、「スタイル」は「標準」という段落スタイルが適用されています。
1. 「行の高さ」の調節方法  
フォントやサイズを変更したときに、「行間」が意図せず広くなって、詰めたいけれど詰められない、どこを設定すれば行間が変えられるのかわからない、という経験をしたことはないでしょうか。  
Word において、行間に影響を与える設定には以下があります。  
・「ページ設定」->「文字数と行数」->「行数」、「行送り」  
・「スタイル」->「間隔：段落前、段落後」  
・「スタイル」->「間隔：行間」  
※補助的に利用するとよい機能：「ページ設定」->「その他」->「行番号」->「行番号を追加する」
  
- 「行送り」：  
「ページ設定」->「文字数と行数」->「行数」、「行送り」
「行数」を増減させると、連動して「行送り」（1行あたりの高さ）が調節されます。  
逆も同様で、「行送り」を増減させたら、連動して「行数」が調節されます。  
「行数」を最小値の「1行」としてみると、「行送り」が「657.6pt」となり、「2行」とするとその半分の「328.8pt」になります。以下例えば「10行：65.75pt」、「20行：32.85pt」、「30行：21.9pt」、「40行：16.4pt」となり、最大値の「45行」とすると、「14.6pt」となります。  
「行数」をあまりに多くしていくと、フォントの高さが「行送り」の高さを超えてしまうことがあります。  
そうなると、1行が2行とみなされて、１ページに表示される実際の行数が、指定した「行数」の半分となってしまいます。  
その場合、「行数」を減らしてみるか、「フォントサイズ」を小さくすると、指定したとおりの行数となります。（ほかの設定のせいで、そうならない場合もあり。）  
  
- 「行間」：  
1文が長くて行末に達した場合の、折り返した次の行との高さの間隔。
「段落スタイル」または「リンク（段落と文字）」の「変更」->「書式」->「段落」->「インデントと行間隔」->「間隔：行間」で、「１行」、「１．５行」「２行」など。  
  
- 「段落前」、「段落後」：  
「Enter」で改行した場合、厳密には「改行」ではなく、実は「改段落」である。  
次の行は「別の段落」ということになり、「段落後」と「段落前」という「間隔」が発生します。  
「段落スタイル」または「リンク（段落と文字）」の「変更」->「書式」->「段落」->「インデントと行間隔」->「間隔：段落前、段落後」で、「０．５行」「１行」など。  
「スタイル」のうち、「段落スタイル」は、段落単位で指定できるスタイルです。
前後の「段落」が同じ「段落スタイル」の場合は、段落間にスペースを追加しない、という設定もできます。
段落を改めずに改行のみするには、「Shift + Enter」を入力します。  
こうすることで、行間を「行間」の場合と同等にできます。  


なお、新規文書で特に「スタイル」を指定していない場合、「本文」レベルに紐づけられた「標準」スタイルが適用されていることになります。  
- 「文字数と行数」  
「ページ設定」->「文字数と行数」->「行数」、「行送り」
スタイルの「間隔：行間」や「間隔：段落前、段落後」で１．５行や０．５行などと指定してあると、１行が１．５行扱いになったり、改段落で０．５行の行間が挿入されたりして、その分、そのページに収まる文字の行数が減ってしまう。
- 「フォント」の高さによる挙動に注意  
フォントの種類ごとに、文字の高さがあり、これが１行あたりの高さの設定値を超えてしまうと、「１行が２行とカウントされ」て、想定とはかけ離れた行間となってしまうことがある。  
文字の大きさからして、1ページあたり何行ほど入るとバランスがよいかを見て、割り戻した値を設定する必要がある。  
- リボンUIの「段落」などで設定変更した場合、その設定が適用されるのはいま選択してる段落のみとなってしまう。文書全体に設定が適用されるようにするには、「スタイル」の「段落」で設定変更すること。

### アウトライン機能を使ってみる
1. 「アウトライン」とは、文書の段落の見出しにレベルをつけて、章・節・項などの階層状に文書を構造化する機能。
2. 階層は上からレベル１（章）、レベル２（節）、レベル３（項）など。
3. 階層が同じであれば、同じ粒度の見出しがついているかを確認し、必要に応じてレベルを上げたり下げたりして揃えていく。
4. 各見出しや各段落の並び順序が適切かを確認し、必要に応じて前後を入れ替えたりして整えていく。

### スタイル機能を使ってみる
1. 「スタイル」とは、複数の書式設定をセットにして一括指定する機能。
2. よく使うスタイルは「表題」「見出し１」「見出し２」「見出し３」「本文」「強調太字」など。
3. 文書に書式設定したい場合に、その範囲を選択して書式をひとつひとつ設定するのではなく、「見出し」なのであれば「アウトライン」で「レベル」を指定し、「本文」なのであれば「スタイル」の「本文」や「強調太字」などを指定する。

### 「文書全体」または「セクション」に対する設定

### 「段落」または「文字列」に対する設定
- 「スタイル」
  - 「スタイル」を特に指定しなくても、実は「標準」というスタイルが適用されている。したがって、「特になにも書式設定をしなかった場合のフォントや文字サイズを指定したい場合」には、「標準」スタイルの設定を詰めればよい。また、アウトラインでレベルを指定しなかった場合（「本文」レベルを指定した場合）に適用されるが、これは「本文」レベルに対して「標準」スタイルもしくは「本文」スタイルが適用されているから。（「本文」というスタイルの「基準のスタイル」が「標準」スタイルとなっている。）
- 「スタイル」で設定できるもの
  - 「次の段落のスタイル」：改行（Enter）したときは、編集記号「カギ矢印」で、「段落」が変わる。このとき、次の段落に、ここで指定した「スタイル」が適用される。「段落」を変えずに改行だけするには、（Shift + Enter）で、編集記号は「下向き矢印」となる。
  - 「フォント」
  - 「段落」：他の段落にも反映される、「スタイル」に対する「段落」の書式設定と、その場かぎりにしか反映されない「いま選択している段落」に対する「段落」の書式設定とがあるので、意識して区別する必要がある。
  - 「タブとリーダー」
  - 「罫線と網かけ」
  - 「箇条書きと段落番号」
  - 「文字の効果」
  - 「クイックスタイル」：リボンのスタイル欄にそのスタイルが表示されて、呼び出しやすくなる。（クイックスタイルギャラリー）
- 「他の文書でもこのスタイルを使用する」：「スタイル」の設定を詰めたら、その設定を他の文書でも適用できる。逆に、このチェックボックスにチェックを入れないと、せっかくスタイルの設定を詰めても、他の文書ではデフォルト設定に戻ってしまう。
- デフォルトの「スタイル」のうちカスタマイズするもの
  - 「標準」：最初の段落の本文の、初期の書式設定の一式。
  - 「見出し１」：アウトラインのレベル１に定義する。章に相当。
  - 「見出し２」：アウトラインのレベル２に定義する。節に相当。
  - 「見出し３」：アウトラインのレベル３に定義する。項に相当。
  - 「表題」：文書の表紙や１枚目の題名。
  - それ以外によく使う「スタイル」があれば、適宜、カスタマイズする。
